Summary

A table of the characteristics of the data used by JpsonicComparatorsTestUtils. It is a documented version of what was previously described in annotations. (If changes and improvements are made frequently, annotations are easier to manage than documenting.) However, the data in this class contains special strings and is documented because the method is unsuitable.

The tests in this class are to create a dataset that covers the range of character patterns that are easy to use for the characters of songs in Japanese, and to check whether they can be sorted correctly.
